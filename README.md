## Overview

This is my personal portfolio website built to showcase my projects, technical skills, and academic background in Artificial Intelligence and Data Science. The site reflects my journey as a developer and highlights the applications and systems I have worked on.

The portfolio is designed to be clean, responsive, and easy to navigate, focusing on clarity and real project demonstration rather than visual clutter.

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- Node.js
- Git & GitHub
- Vercel (Deployment)

---

## Key Features

- Responsive design for mobile and desktop
- Dedicated sections for projects, skills, and resume
- Clean UI with minimal design
- Easy-to-update component structure
