# Personal Portfolio

This repository contains the source code for my **personal portfolio website**. The portfolio is designed to showcase my **projects, certifications, and development journey**, along with my interest in **Web Development and Artificial Intelligence**.

It serves as a central platform to present my work, technical skills, and learning progress in a structured and professional way.

The website focuses on a **clean interface, responsive layout, and smooth navigation** to provide a simple and user-friendly browsing experience.

---

## Features

- Responsive and modern user interface  
- Projects section highlighting development work  
- Certifications and achievements display  
- Resume section for professional details  
- Blog section for sharing knowledge and learning  
- Smooth navigation and organized layout  

---

## Installation

Clone the repository

```bash
git clone https://github.com/Ojaswimakode07/portfolio.git
```

Navigate to the project directory

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm start
```
